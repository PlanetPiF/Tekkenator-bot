# Tekkenator-bot
A discord bot for Tekken 7.

Technologies used:

Java

Spring Boot

Spring Data JPA and Hibernate - Database access

Apache Log4J - logging

Java Discord Api (JDA) - https://github.com/DV8FromTheWorld/JDA

MySql + connector - database
